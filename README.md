===========================================================
      PROJECT: FRUIT SHOP - Full-Stack Web Application
===========================================================

1. ข้อมูลเบื้องต้นเกี่ยวกับโครงการ
------------------------------
Fruit Shop เป็นเว็บไซต์อีคอมเมิร์ซสำหรับจำหน่ายผลไม้ออนไลน์ 
พัฒนาด้วยสถาปัตยกรรมแบบ MVC (Model-View-Controller) เพื่อความเป็นระเบียบของโค้ด
มีการออกแบบในสไตล์ Frutiger Aero ที่เน้นความสดใสและสวยงาม

2. เทคโนโลยีที่ใช้ (Tech Stack)
---------------------------
- ส่วนหน้าบ้าน (Frontend): HTML, CSS, EJS Template Engine
- ส่วนหลังบ้าน (Backend): Node.js, Express Framework
- ฐานข้อมูล (Database): PostgreSQL (ผ่านระบบ Cloud ของ NeonDB)
- ระบบสมาชิก: express-session และการเข้ารหัสรหัสผ่านด้วย bcrypt

3. โครงสร้างไฟล์และโฟลเดอร์ (Folder Structure)
------------------------------------------
Fruits-Shop/
├── config/             -> ตั้งค่าการเชื่อมต่อฐานข้อมูล (db.js)
├── controller/         -> ควบคุม Logic การทำงานของร้านค้า (shopController.js)
├── model/              -> เขียนคำสั่ง SQL เพื่อจัดการข้อมูลใน Database (shopModel.js)
├── routes/             -> จัดการเส้นทาง URL ของเว็บไซต์ (shopeRoutes.js)
├── views/              -> ส่วนแสดงผลหน้าเว็บ (EJS Template)
│   ├── partials/       -> ส่วนประกอบที่ใช้ซ้ำ (header, navbar, footer)
│   ├── index.ejs       -> หน้าหลัก (Home)
│   ├── categories.ejs  -> หน้าหมวดหมู่สินค้า
│   ├── cart.ejs        -> หน้าตะกร้าสินค้า
│   ├── login.ejs       -> หน้าเข้าสู่ระบบ
│   └── register.ejs    -> หน้าสมัครสมาชิก
├── public/             -> ไฟล์ที่ให้ Browser เข้าถึงได้โดยตรง
│   ├── css/            -> ไฟล์ตกแต่งหน้าเว็บ (style.css)
│   ├── js/             -> ไฟล์ลูกเล่นฝั่ง Frontend (main.js)
│   └── images/         -> รูปภาพสินค้าและไอคอนทั้งหมด
├── app.js              -> ไฟล์หลักสำหรับรัน Server
├── .env                -> ไฟล์เก็บค่าความลับ เช่น DATABASE_URL (ไม่ได้รวมใน Source Code)
├── .gitignore          -> ระบุไฟล์ที่ไม่ต้องการนำขึ้น Git (เช่น node_modules)
├── package.json        -> รายการ Library และ dependencies ทั้งหมด
└── readme.txt          -> คำอธิบายโครงการและวิธีติดตั้ง

4. วิธีการรันระบบ (How to Run)
--------------------------
หากต้องการรันระบบในเครื่อง ให้ทำตามขั้นตอน:

1. ติดตั้งโปรแกรมที่จำเป็น:
   - ตรวจสอบว่าในเครื่องมี Node.js ติดตั้งอยู่

2. การติดตั้ง Dependencies:
   - เปิด Terminal ในโฟลเดอร์โครงการ แล้วพิมพ์คำสั่ง:
     npm install

3. การตั้งค่าเกี่ยวกับฐานข้อมูล:
   - ตรวจสอบไฟล์ใน config/db.js ให้แน่ใจว่าเชื่อมต่อกับ Database ถูกต้อง
   - ข้อมูลสินค้าถูกจัดเก็บไว้บน Cloud ของ NeonDB เรียบร้อย

4. การเริ่มทำงานของ Server:
     npm run dev

5. เข้าใช้งานหน้าเว็บ:
   - เปิด Browser แล้วไปที่: http://localhost:3000
